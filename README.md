# Poisoned Potato
```Проект по дисциплине ОП.03 / Сапёр Огородник```

### Какую проблему решает проект:
Данный проект основывается на логической компьютерной игре головоломке - "сапёр". Сама по себе классическая игра разгружает мозг от постоянного нахождения в условиях информационной перегрузки, наша игра направленна на создание такого же медативного процесса только для любителей огорода и поездок за город.

### Целевая аудитория: 
Офисные работники мечтающие о даче и полях.

### Цели
Данный проект нацелен на создание игры, схожей с принципом игры "Сапёр", и добавления в эту игры новых интерактивных режимов игры. Классический игровой процесс заключается в разминировании поля, открытия ячеек не содержащих мину. Этот учебный проект переформирует само понятие игры основаное на разминировании поля, задавая атмосферу жарких знойных деньков на даче. В моем проекте, помимо реализации похожего на игру "Сапёр" игрового процесса, вы поучаствуете в процессе посева и сбора урожая картофеля. Процесс сбора будет являться основным режимом игры, на поле вам нужно будет найти потенциальный отравленный картофель. А процесс посева - это маленькое дополнение к основному игровому процессу, где вы будете сеять картофель при помощи краткого теста: выбора условий для посадки поля картофеля.
